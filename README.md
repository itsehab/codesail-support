## FAQ

**How do I pair my phone with my computer?**
1. Install the CLI: `npm install -g codesail`
2. Run `codesail auth login` in your terminal
3. Scan the QR code with CodeSail on your iPhone

**Do I need a server?**
No. CodeSail connects to a managed relay server by default. You can optionally configure a self-hosted server in Settings.

**Is my data encrypted?**
Yes. All messages are end-to-end encrypted on your device before being sent. The relay server never sees your message content.

**What integrations are supported?**
Vercel, GitHub, OpenAI, Sentry, Neon, Resend, AWS, and Google AI.
